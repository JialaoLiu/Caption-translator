# Bilibili 直播姬使用说明

这个工具不是 OBS 插件。它会打开一个独立的置顶字幕窗口，直播姬通过素材捕捉这个窗口。

推荐链路：

```text
字幕工具 pinned 窗口
↓
Bilibili 直播姬
↓
窗口捕捉 / 截屏捕捉
```

## 方法一：窗口捕捉

1. 打开字幕工具。
2. 点击 `Start`。
3. 在直播姬中添加直播素材。
4. 选择 `窗口捕捉`。
5. 捕捉窗口名为 `Caption Translator Subtitle` 的字幕条窗口。
6. 在直播画面中调整字幕位置和大小。

## 方法二：截屏捕捉

如果窗口捕捉不稳定：

1. 在直播姬中添加直播素材。
2. 选择 `截屏捕捉`。
3. 框选字幕窗口所在区域。
4. 直播过程中不要移动字幕窗口，除非重新框选区域。

## 性能建议

直播打 PUBG 时建议：

- 默认 `small + CPU + int8`
- CPU 占用高时切换到 `base` 或 `tiny`
- 不默认使用 CUDA
- 不默认加载 `medium` 或 `large-v3`
- 字幕刷新间隔保持 `2` 到 `3` 秒

## 备用 txt 输出

工具会把当前字幕写到项目根目录的 `subtitle.txt`。如果后续需要给其他直播软件或字幕插件读取，可以使用这个文本文件作为备用输出。
